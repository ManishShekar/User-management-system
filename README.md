# User Management System

## Description
A full stack web application developed using Java, J2EE (Servlets), and JDBC.
The application performs CRUD operations on user data stored in a MySQL database.

## Tech Stack
- Java
- J2EE (Servlets)
- JDBC
- MySQL
- HTML, CSS
- Apache Tomcat

## Features
- Add new users
- View all users
- Update user details
- Delete users
- JDBC-based database interaction

## Project Structure
- player/src/main/java → Servlets, DAO, Model classes
- WebContent → HTML, CSS files

## How to Run
1. Clone the repository
2. Import into Eclipse / IntelliJ
3. Configure MySQL database
4. Deploy on Apache Tomcat
